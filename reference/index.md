# Package index

## Main function

Launch the interactive annotation gadget

- [`ggannotate()`](https://mattcowgill.github.io/ggannotate/reference/ggannotate.md)
  : Interactively annotate a ggplot2 plot

## Helper functions

Functions for programmatic annotation building

- [`make_layer()`](https://mattcowgill.github.io/ggannotate/reference/make_layer.md)
  : Create a ggplot2 annotation given some aesthetics, parameters, and
  facet variables + values.
